# Практична 1 — Базовий клієнт – бекенд – БД

## Тема проєкту
Міні онлайн-магазин аксесуарів.

## Реалізовано
- Обрано тему проєкту
- Піднято backend на Node.js + Express
- Підключено локальну БД SQLite
- Створено таблицю items + тестові дані
- Реалізовано endpoint GET /items
- Створено просту HTML-сторінку
- Клієнт викликає API та відображає список товарів

## Запуск
### 1. Backend
cd backend
npm install
node server.js

### 2. Frontend
Відкрити `client/index.html` у браузері

## Результат
Натискаємо "Load items" → бачимо список товарів з БД.
