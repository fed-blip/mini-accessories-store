# Міні онлайн-магазин аксесуарів

## Короткий опис системи
Міні онлайн-магазин аксесуарів — це веб‑застосунок, що дозволяє користувачам переглядати каталог аксесуарів, отримувати інформацію про товари та надалі оформлювати замовлення.  
Система розробляється поетапно в рамках навчального курсу та буде поступово розширювати свій функціонал: від статичного каталогу до повноцінного магазину з базою даних, авторизацією та контейнеризацією.

Мета системи — продемонструвати вміння працювати з веб‑технологіями, бекенд‑сервером, базою даних, документацією та DevOps‑інструментами.

---

## Ролі користувачів
1. **Гість (незареєстрований користувач)**  
   - може переглядати каталог
2. **Користувач (у майбутньому)**  
   - зможе оформлювати замовлення
3. **Адміністратор (у майбутньому)**  
   - керує товарами (CRUD)
   - переглядає статистику

---

## Основні сценарії використання
1. Користувач заходить на сайт та переглядає список аксесуарів.
2. Користувач відкриває сторінку товару і бачить деталі.
3. (майбутнє) Користувач додає товар у кошик.
4. (майбутнє) Користувач оформляє замовлення.
5. (майбутнє) Адміністратор додає новий товар.

---

## Функціональні вимоги
✔ Відображення каталогу товарів  
✔ Отримання списку товарів через backend API  
⬜ Перегляд окремого товару  
⬜ Авторизація користувачів  
⬜ CRUD для адміністрування товарів  
⬜ Оформлення замовлення

---

## Нефункціональні вимоги
- Система має працювати стабільно
- Простота використання (зрозумілий інтерфейс)
- Масштабованість — можливість додати нові модулі
- Безпека — захист даних користувачів
- Швидкість — сторінки повинні відкриватися без затримок
- Можливість розгортання в Docker
