# Glossary – Ubiquitous Language

**Item** — товар у магазині, який можна переглянути та купити  
**Category** — група товарів  
**User** — користувач системи  
**Order** — замовлення, яке містить один або кілька товарів  
**OrderItem** — конкретний товар у замовленні  
**Payment** — процес оплати замовлення  
**Cart** — тимчасовий список товарів перед оформленням замовлення
